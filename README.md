# Portfolio Website

## Project Overview
This is a portfolio website showcasing the projects and works of Ganesh. The website features various projects that highlight skills in web development and design.

## Setup Instructions
1. **Clone the repository**: 
   ```bash
   git clone https://github.com/gampaganesh04-code/ganesh18.git
   ```
2. **Navigate into the project directory**: 
   ```bash
   cd ganesh18
   ```
3. **Install dependencies**: 
   ```bash
   npm install
   ```
4. **Run the application**: 
   ```bash
   npm start
   ```

Visit `http://localhost:3000` in your browser to see the portfolio website in action.
